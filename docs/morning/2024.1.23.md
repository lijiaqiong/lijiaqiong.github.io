【工具使用】
Java 内存模型（Java Memory Model，JMM）就是一种符合内存模型的规范，解决由于多线程通过共享内存进行通信时，存在的本地内存数据不一致、编译器会对代码指令重排序、处理器会对代码乱序执行等带来的问题。目的是保证并发编程场景中的原子性、可见性和有序性。
